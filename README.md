# C++ & Win32 API 기반 디지털 영상처리 기초 프로젝트

## 📌 프로젝트 개요
이 프로젝트는 외부 라이브러리 없이 C++와 Win32 API만을 사용하여 구현한 **기초 디지털 영상처리 프로그램**입니다.
RAW 포맷의 영상 데이터를 직접 제어하며 화소 점 처리, 영역 처리, 기하학적 변환 등 영상처리의 핵심 알고리즘을 직접 구현하여 원리를 학습하는 데 중점을 두었습니다.

## 🛠️ 개발 환경
- **OS**: Windows 10/11
- **Language**: C++
- **IDE**: Visual Studio 2022
- **Library**: Win32 API (GDI)

---

## 🧩 주요 기능

### 1. 화소 점 처리 (Point Processing)
- **산술 연산**: 덧셈(밝기 증가), 뺄셈(밝기 감소), 곱셈, 나눗셈
- **논리 연산**: 이진화(Binarization), 반전(Inverse)
- **비선형 변환**: 감마 보정(Gamma Correction), 파라볼라(Parabola) 변환

### 2. 기하학적 변환 (Geometric Transformation)
- **확대/축소**: 기본 역방향 사상(Backward Mapping) 알고리즘 적용
- **이동**: X, Y 축 평행 이동
- **대칭**: 좌우, 상하, 상하좌우 미러링(Mirroring)
- **회전**: 삼각함수를 이용한 영상 회전

### 3. 화소 영역 처리 (Spatial Filtering)
3x3 마스크(커널) 연산을 이용한 필터링 구현
- **엠보싱(Embossing)**
- **블러링(Blurring)**
- **샤프닝(Sharpening)**
- **에지 검출(Edge Detection)**: 기본 에지, 라플라시안 등

---

## 📂 코드 구조

### 핵심 구성
- **입출력**: `fread`, `fwrite`를 이용한 RAW 파일 처리. 헤더가 없는 데이터를 다루므로 파일 크기의 제곱근으로 해상도를 계산합니다.
- **메모리 관리**: 2차원 배열 포인터(`unsigned char**`)를 사용하여 영상 데이터를 동적으로 할당 및 해제합니다.
- **렌더링**: Win32 API의 `StretchDIBits` 함수를 사용하여 처리된 영상 데이터를 윈도우 창에 출력합니다.
- **인터페이스**: `_kbhit()` 함수를 이용한 콘솔 기반의 메뉴 시스템을 통해 사용자의 입력을 실시간으로 처리합니다.

---

## 🚀 사용 방법

1. **실행**: 프로그램을 실행하면 콘솔 메뉴와 빈 GUI 창이 나타납니다.
2. **이미지 로드**: 메뉴에서 `1`을 입력하고 `.raw` 이미지 파일명을 입력합니다.
3. **기능 수행**:
   - `A`: 밝기 조절
   - `L`: 영상 확대
   - `N`: 영상 회전
   - `S`: 엠보싱 필터 적용
   - `9`: 프로그램 종료
4. **결과 확인**: 기능 수행 후 GUI 창에서 처리된 영상을 즉시 확인할 수 있습니다.
